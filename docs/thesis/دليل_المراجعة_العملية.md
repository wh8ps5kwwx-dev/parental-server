# دليل المراجعة العملية — MY Rana v1.3

## قبل البدء
- جوال ولي الأمر: APK `MYRana-Parent-ولي-الأمر.apk`
- جوال الطفل: APK `MYRana-Child-طفل.apk`
- بريد Gmail واحد لولي الأمر
- إنترنت شغّال (جربي 4G إن فشل Wi‑Fi)
- أيقظي السيرفر: افتحي في Chrome  
  `https://parental-server-4mms.onrender.com/health`

## المرحلة 1 — الربط (TC-01 → TC-03) ✅ مُختبر سابقاً

| الخطوة | الجهاز | ماذا تفعلين | لقطة للبحث |
|--------|--------|-------------|------------|
| 1 | ولي الأمر | بريد Gmail + OTP الرسالة 1 | شكل 5.3 |
| 2 | الطفل | تسجيل ببريد الأم → CHILD بالبريد | شكل 5.1 + 5.2 |
| 3 | ولي الأمر | CHILD + OTP الرسالة 3 | شكل 5.3 |
| 4 | الطفل | انتظار الربط → صلاحيات → أكاديمية | شكل 5.6 |

**نجاح:** تظهر لوحة التحكم بدون أخطاء.

## المرحلة 3 — تنبيه كلمة خطرة (TC-05) ⚠️ مطلوب

**على جوال الطفل:**
1. تأكدي: صلاحية **Accessibility** مفعّلة لتطبيق أكاديمية العباقرة.
2. افتحي **Chrome** أو تطبيق بحث.
3. ابحثي عن: `قمار` أو `casino`
4. انتظري 15–30 ثانية.

**على جوال ولي الأمر:**
5. افتحي لوحة التحكم → التنبيهات.
6. يجب ظهور تنبيه عن كلمة خطرة.

📸 **لقطة:** شكل 5.7 (طفل + أم)
