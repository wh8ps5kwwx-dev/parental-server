## الطريقة 2 — APK كامل (Buildozer + WSL)

> WSL غير مثبت على جهازك حالياً — تحتاجين تثبيته أولاً.

### أ) تثبيت WSL (مرة واحدة — من PowerShell كمسؤول)

```powershell
wsl --install
```

أعيدي تشغيل الكمبيوتر → افتحي **Ubuntu** من قائمة ابدأ.

### ب) داخل Ubuntu

```bash
sudo apt update
sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config zlib1g-dev libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev

pip3 install buildozer cython

cd /mnt/e/parent_monitor_project/mother-app
buildozer android debug
```

### ج) الناتج

```
bin/myrana_mother-1.0.0-arm64-v8a_debug.apk
```

أرسليه لجوال الأم → ثبّتي → افتحي **MYRana Mother**.
