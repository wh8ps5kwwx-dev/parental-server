# MY Rana — تطبيق الأم على الجوال (Python)

حزمة **myrana_mother_phone.zip** = مشروع متكامل يعمل على **Pydroid 3** بدون حاجة لمجلد المشروع الكامل.

## محتويات الحزمة

| الملف | الوظيفة |
|-------|---------|
| `main.py` | واجهة Kivy كاملة |
| `guardian_api.py` | اتصال السيرفر (بريد، ربط، حظر، تقارير) |
| `Arabic.ttf` | خط عربي |
| `requirements.txt` | المكتبات المطلوبة |
| `README_PHONE.md` | هذا الملف |

## التثبيت على جوال الأم

1. ثبّتي **Pydroid 3** من Play Store
2. فكّي الضغط عن `myrana_mother_phone.zip` في مجلد مثل `Pydroid3/projects/myrana/`
3. من Pydroid → **Pip** ثبّتي:
   ```
   kivy
   requests
   arabic-reshaper
   python-bidi
   ```
4. افتحي `main.py` → **تشغيل ▶**

## خطوات الربط

1. **جوال الطفل:** ثبّتي APK الطفل → سجّلي الجهاز → انسخي `CHILD-...`
2. **تطبيق الأم:** بريدك → رمز التحقق من Gmail
3. **ربط طفل:** الصقي كود الطفل → إرسال رمز الربط → أدخلي الرمز → حفظ
4. **حظر:** اكتبي اسم التطبيق (مثل Granny) → حظر تطبيق

## السيرفر

- https://parental-server-4mms.onrender.com
- يجب أن يكون السيرفر شغّال ومتصل بالإنترنت
